# sorbet-cfg

Powerful semantic analysis using Sorbet's CFGs

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'sorbet-cfg'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install sorbet-cfg
